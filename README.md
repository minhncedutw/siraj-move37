## Installation

### Virtualenv

OS X & Linux:

```sh
python -m venv environment
source environment/bin/activate
pip install -r requirements.txt
```

Windows:

```sh
python -m venv environment
environment\Scripts\activate.bat
pip install -r requirements.txt
```

### Anaconda/Miniconda

Windows & OS X & Linux:

```sh
conda env create -f environment.yml
```
